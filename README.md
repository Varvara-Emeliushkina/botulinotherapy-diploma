# Автоматизация процесса формирования протокола ботулинотерапии

Выпускная квалификационная работа (бакалавр)

## О проекте
Разработка клиент-серверного приложения для автоматизации учета процедур 
ботулинотерапии по заказу врача-невролога Федерального центра нейрохирургии 
(г. Новосибирск).

## Технологический стек
- **Backend:** Java, Spring Boot, Spring Security (JWT), Spring Data JPA
- **Database:** MySQL
- **Client:** Flutter (Dart)
- **Build tools:** Maven/Gradle

## Структура репозитория

mobile/ # мобильное приложение (Flutter)
